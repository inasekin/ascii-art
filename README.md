# ASCII Art Converter

Конвертер растровых изображений в текстовый ASCII-арт.

## Требования

- Python 3.12+
- [uv](https://github.com/astral-sh/uv)

## Установка и запуск

```bash
uv sync
uv run python main.py
```

## Поддерживаемые форматы

PNG, JPG, JPEG, BMP, GIF

## Возможности

- Три набора символов: стандартный, расширенный, градиентный
- Инверсия яркости
- Цветной ASCII-арт с ANSI-кодами
- Предпросмотр в реальном времени (GUI)
- Пакетная обработка нескольких изображений
- Сохранение в .txt и копирование в буфер обмена
- Автосохранение настроек
