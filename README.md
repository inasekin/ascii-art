# Конвертер изображений в ASCII-арт

Десктопное приложение на Python/tkinter, преобразующее растровые изображения
в текстовое ASCII-представление с сохранением истории в SQLite.

## Возможности

- Форматы входных изображений: PNG, JPEG, BMP, GIF (с анимацией)
- Три набора символов: Standard / Extended / Gradient
- Инверсия яркости, цветной ANSI-вывод
- Анимация вращения изображения (360°)
- Пакетная обработка нескольких файлов
- Сохранение результата в `.txt`, копирование в буфер
- История конвертаций (SQLite база данных)

## Требования

| Компонент | Версия |
|-----------|--------|
| Python    | 3.10+  |
| Pillow    | 9.0+   |
| tkinter   | входит в Python |
| sqlite3   | входит в Python |

## Установка и запуск

### Вариант 1 — через uv (рекомендуется)

```bash
git clone <url>
cd ascii-art
uv sync
uv run python main.py
```

### Вариант 2 — через pip

```bash
pip install Pillow
python main.py
```

## Инструкция по использованию

1. **Открыть файл** — кнопка «Открыть…», выбрать PNG/JPG/BMP/GIF.
2. **Настроить параметры**:
   - *Ширина* — количество символов в строке (10–300).
   - *Высота авто* — сохранить пропорции; снять галку для ручной высоты.
   - *Набор символов* — Standard (по умолчанию), Extended, Gradient.
   - *Инверсия яркости* — поменять тёмное/светлое местами.
   - *Цветной (ANSI)* — цветной вывод в окне предпросмотра.
   - *Вращать изображение* — создать 360°-анимацию.
3. **Конвертировать** — нажать кнопку или изменить любой параметр.
4. **Сохранить** — «Сохранить в .txt…» или «Копировать в буфер».
5. **История** — кнопка «История» откроет таблицу последних конвертаций.
6. **Пакетная обработка** — добавить несколько файлов → «Пакетная обработка…».

## Структура проекта

```
ascii-art/
├── main.py              # Точка входа
├── gui.py               # Графический интерфейс (tkinter)
├── converter.py         # Алгоритм конвертации (Pillow)
├── settings.py          # Настройки приложения (JSON)
├── database.py          # Модуль SQLite (история конвертаций)
├── queries.sql          # SQL-запросы (5 штук)
├── subject_area.md      # Описание предметной области
├── er_diagram.tex       # ER-диаграмма (LaTeX/TikZ, нотация Чена)
├── logical_diagram.tex  # Логическая схема БД (LaTeX/TikZ)
├── presentation.tex     # Презентация (LaTeX Beamer)
├── report.tex           # Пояснительная записка (LaTeX)
├── ascii_art.db         # База данных SQLite (создаётся при запуске)
└── settings.json        # Настройки (создаётся при запуске)
```
